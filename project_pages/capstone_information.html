Graph-based Blockchain Visualization

Fall 2022 - Spring 2023

GitHub

This project takes the Elliptic dataset and graphs smaller subsets in a full-stack web application. The Elliptic dataset is a large collection of over 200,000 Bitcoin transactions that occurred over a period of 50 time steps. The front end was created using React JS as the framework, D3.js for the graph visualization and Material UI for the UI components. The backend consists of a Neo4j graph database as well as an API made using node.js and Express.js. The GitHub link is currently private due to it being an ongoing school project, but if interested, reach out and I will be glad to send a copy of the code or give limited access to the GitHub repository.
