Android Library App

An open source book collection app for Android that allows a user to catalog their collection of books in a simple and minimalist app

Key Features

  • Book details are pulled from the OpenLibrary API
  • Saved books are stored in a local SQLite database
  • Personal library can be searched by title, author, or isbn
  • A book can be removed from the collection on its details page
  • Multithreading is used to speed up retrieving book details from the internet
